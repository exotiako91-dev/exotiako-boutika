# ExoTiako v2.0

Version statique compatible GitHub Pages.

## Tester en local

```bash
python3 -m http.server 8000
```

Puis ouvrir : http://localhost:8000

## Déployer

Copiez tout le contenu à la racine du repo, puis :

```bash
git add .
git commit -m "Version 2.0 - filtres et beignet sucre"
git push origin develop-v2.0
```
